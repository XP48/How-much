# **How Much**

## **How Much** is a web calendar in order to preview the spended time and the remaining time

### Legend of boxes
- Time spent in lightgray
- Now in green
- Time remaining in gray
- Marked case in red

### Available sections / features
- Year
- Month
- Week
- Day
- *memento mori*

### How to mark a box ?
To mark a case click on a gray box !
